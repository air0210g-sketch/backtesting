# 数据下载与工具集 (Data Tools)

本目录集成了该项目所有与数据获取、更新、校验相关的脚本。

## 1. 核心下载脚本
| 脚本名称                 | 功能描述            | 使用场景                                                    |
| :----------------------- | :------------------ | :---------------------------------------------------------- |
| `download_all_stocks.py` | **批量下载 (推荐)** | 从 `stock_list.json` 读取标的清单，自动进行全量或增量下载。 |
| `download_candles.py`    | **单只/自选下载**   | 下载单只股票或特定 Longport 自选组的数据。                  |
| `update_data_history.py` | **增量更新**        | 扫描 `stock_data/` 目录并更新现有文件的缺失历史。           |

## 2. 数据校验脚本
| 脚本名称               | 功能描述           | 使用场景                                 |
| :--------------------- | :----------------- | :--------------------------------------- |
| `verify_batch_data.py` | **随机采样校验**   | 随机抽取 10 个文件校验格式、行数和空值。 |
| `verify_data.py`       | **深度完整性校验** | 检测数据时间步长对齐、重复值等问题。     |

---
## 使用方法
建议在项目根目录下执行脚本，例如：
```bash
python3 scripts/data_tools/download_all_stocks.py
```
所有数据将统一存储在 `stock_data/` 目录下。
